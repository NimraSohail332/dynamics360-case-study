## Project Overview

EventHub is a **single-page application** that helps users discover local events in their area. The application features a clean, modern design with full responsiveness across all devices, showcasing advanced frontend development skills.

### Key Highlights
- **Zero Dependencies** (except Font Awesome for icons)
- **Mobile-First Responsive Design**
- **Modern UI/UX** with glass morphism effects
- **Cross-Browser Compatibility**
- **Advanced Search Functionality**

## Quick Start

### Option 1: Direct File Opening
```bash
1. Download all project files
2. Open index.html in any modern web browser
3. The website loads immediately - no server required!
```

### Option 2: Live Server (VS Code)
```bash
1. Install "Live Server" extension in VS Code
2. Right-click on index.html
3. Select "Open with Live Server"
```

## Responsive Design

| Device | Breakpoint | Optimization |
|--------|------------|--------------|
| **Desktop** | 1200px+ | Full layout with all features |
| **Tablet** | 768px - 1199px | Adapted grid system |
| **Mobile** | 480px - 767px | Stacked layout, hamburger menu |
| **Small Mobile** | < 480px | Optimized for smallest screens |
